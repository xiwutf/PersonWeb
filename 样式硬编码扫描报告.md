## 修复建议

### 优先级 P0 - 立即处理

1. **颜色值硬编码替换**
   - 新增 `--color-primary-gradient` 等渐变变量
   - 新增 `--shadow-primary`、`--shadow-secondary` 等阴影变量
   - 新增 RGBA 色彩变量体系

2. **建议新增变量**：
```css
/* 渐变色变量 */
:root {
  --gradient-primary: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
  --gradient-secondary: linear-gradient(135deg, #f093fb 0%, #f5576c 100%);
}

/* RGBA 阴影变量 */
:root {
  --shadow-primary: 0 0 20px rgba(59, 130, 246, 0.4);
  --shadow-secondary: 0 0 30px rgba(59, 130, 246, 0.6);
  --shadow-tertiary: 0 0 40px rgba(59, 130, 246, 0.8);
}
```

### 优先级 P1 - 近期处理

1. **字体大小规范化**
   - 统一使用 `text-base`、`text-sm` 等变量
   - 移除所有硬编码的 `font-size` 值

2. **间距规范化**
   - 使用 `spacing-2`、`spacing-4` 等间距变量
   - 确保 padding、margin 使用统一变量

3. **圆角规范化**
   - 替换 `4px` 为 `radius-sm`
   - 替换 `8px` 为 `radius-md`
   - 替换 `12px` 为 `radius-lg`

### 优先级 P2 - 后续优化

1. **Z-index 层级整理**
   - 统一使用 `z-modal`、`z-dropdown` 等变量
   - 补充缺失的更高层级变量

2. **内联样式优化**
   - 将复杂的内联样式提取为 CSS 类
   - 简单动态值使用 CSS 变量

## 具体修复示例

### 修复前
```css
.button {
  background: #667eea;
  font-size: 16px;
  padding: 12px 24px;
  border-radius: 8px;
  box-shadow: 0 4px 12px rgba(102, 126, 234, 0.4);

```

```css
.button {
  background: var(--color-primary);
  font-size: var(--text-base);
  padding: var(--spacing-3) var(--spacing-6);
  border-radius: var(--radius-md);
  box-shadow: var(--shadow-primary);
  z-index: var(--z-modal);
}
```
