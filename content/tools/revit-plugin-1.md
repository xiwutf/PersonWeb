# 批量族加载器

## 功能简介

批量族加载器是一款专为Revit用户设计的高效率插件，能够一次性加载多个族文件，大大提升工作效率。

## 主要功能

### 🚀 批量加载
- 支持选择文件夹，一次性加载所有族文件
- 支持递归搜索子文件夹
- 自动跳过已存在的族文件

### 📊 进度监控
- 实时显示加载进度
- 详细的加载日志记录
- 错误文件统计与提示

### 🔧 智能过滤
- 按族类别筛选
- 按文件大小过滤
- 支持自定义文件名模式匹配

## 使用方法

1. 点击Revit插件面板中的"批量族加载器"按钮
2. 选择包含族文件的文件夹
3. 设置过滤条件（可选）
4. 点击"开始加载"按钮
5. 等待加载完成，查看结果报告

## 系统要求

- Revit 2019及以上版本
- Windows 10/11
- .NET Framework 4.8

## 安装说明

1. 下载插件安装包
2. 解压到Revit插件目录
3. 重启Revit
4. 在插件面板中找到并使用
