# ⚡ 内容更新快速参考

## 📝 三步更新法

### 1️⃣ 编辑文件
在 `content/` 目录下找到对应的 Markdown 文件并编辑：

```
content/
├── blog/          # 博客文章
├── tools/         # 工具插件
└── projects/       # 项目展示
```

### 2️⃣ 本地预览
```bash
npm run dev
```
访问 http://localhost:3000 查看效果

### 3️⃣ 构建部署
```bash
npm run build
# 然后将 .output/public 目录部署到服务器
```

## 🎯 常用操作

### 添加新博客文章
```bash
# 1. 创建文件
touch content/blog/new-article.md

# 2. 编辑文件（添加 Front Matter 和内容）

# 3. 预览
npm run dev
```

### 更新工具价格
```bash
# 编辑 content/tools/xxx.md
# 修改 price 字段
price: 29.9  # 新价格
```

### 更新项目状态
```bash
# 编辑 content/projects/xxx.md
# 修改 status 字段
status: "已上线"  # 或 "开发中"
```

## 📝 Front Matter 模板

### 博客文章
```yaml
---
title: "文章标题"
date: "2024-01-20"
tags: ["标签1", "标签2"]
description: "文章描述"
author: "溪午听风"
category: "技术文章"
---
```

### 工具插件
```yaml
---
title: 工具名称
description: 工具描述
price: 19.9
tags: [Revit, 插件]
buy_link: https://example.com
slug: tool-name
date: 2024-01-20
---
```

### 项目展示
```yaml
---
title: "项目名称"
tech: ["Vue.js", "Nuxt 3"]
description: "项目描述"
demo_link: "https://demo.com"
source_link: "https://github.com/..."
slug: "project-name"
date: "2024-01-20"
status: "已上线"
category: "Web应用"
---
```
