# 🖥️ 本地预览指南

本指南将帮助您在本地电脑上预览和测试网站。

---

## 📋 前置要求

### 1. 检查 Node.js 版本

打开 PowerShell 或 CMD，运行：

```bash
node -v
```

**要求**：Node.js 18 或更高版本

如果未安装或版本过低：
- 访问 [Node.js 官网](https://nodejs.org/) 下载安装
- 推荐安装 LTS（长期支持）版本

### 2. 检查 npm

```bash
npm -v
```

npm 通常随 Node.js 一起安装。

---

## 🚀 快速开始（三步）

### 第一步：打开项目目录

在 PowerShell 或 CMD 中，进入项目目录：

```bash
cd D:\00-Project\AI\PersonWeb
```

### 第二步：安装依赖

首次运行或更新依赖后需要执行：

```bash
npm install
```

**说明**：
- 首次运行需要安装所有依赖包
- 如果之前已安装过，可以跳过此步骤
- 安装过程可能需要几分钟，请耐心等待

### 第三步：启动开发服务器

```bash
npm run dev
```

**成功标志**：
```
✔ Nuxt is ready
  ➜ Local:    http://localhost:3000/
  ➜ Network:  http://192.168.x.x:3000/
```

## 🔧 常见问题解决

### ❌ 问题 1：端口被占用

**错误信息**：
```
Error: listen EADDRINUSE: address already in use :::3000
```

**解决方法**：
```bash
# 方法1：关闭占用 3000 端口的程序
# Windows PowerShell
netstat -ano | findstr :3000
taskkill /PID <进程ID> /F

# 方法2：使用其他端口
# 在 nuxt.config.ts 中添加：
export default defineNuxtConfig({
  devServer: {
    port: 3001  // 改为其他端口
  }
})
```

### ❌ 问题 2：依赖安装失败

**错误信息**：
```
npm ERR! code ERESOLVE
```

**解决方法**：
```bash
# 清除缓存
npm cache clean --force

# 删除 node_modules 和 package-lock.json
rm -r node_modules
rm package-lock.json

# 重新安装
npm install
```

### ❌ 问题 3：Node.js 版本过低

**错误信息**：
```
The engine "node" is incompatible with this module
```

**解决方法**：
1. 访问 [Node.js 官网](https://nodejs.org/)
2. 下载并安装 Node.js 18 或更高版本
3. 重启终端，重新运行 `npm install`

### ❌ 问题 4：页面显示空白

**可能原因**：
1. 开发服务器未正常启动
2. 浏览器缓存问题
3. 代码有错误

**解决方法**：
```bash
# 1. 检查终端是否有错误信息
# 2. 清除浏览器缓存（Ctrl + Shift + Delete）
# 3. 尝试无痕模式打开
# 4. 检查浏览器控制台（F12）是否有错误
```

### ❌ 问题 5：修改内容不更新

**解决方法**：
```bash
# 1. 确认文件已保存（Ctrl + S）
# 2. 检查文件路径是否正确
# 3. 重启开发服务器（Ctrl + C 停止，然后 npm run dev）
# 4. 清除浏览器缓存
```

## 💡 开发技巧

### 1. 使用 VS Code 编辑

**推荐扩展**：
- **Vue Language Features (Volar)** - Vue 3 支持
- **Markdown Preview Enhanced** - Markdown 预览
- **Tailwind CSS IntelliSense** - Tailwind 自动补全

### 2. 浏览器开发者工具

按 `F12` 打开开发者工具：
- **Console**：查看错误和日志
- **Network**：查看网络请求
- **Elements**：检查 DOM 结构

### 3. 实时编辑测试

1. 打开两个窗口：
   - 一个显示网站（http://localhost:3000）
   - 一个编辑代码（VS Code）

2. 修改代码并保存
3. 观察浏览器自动刷新

### 4. 移动端预览

开发服务器启动后，会显示：
```
Network:  http://192.168.x.x:3000/
```

在手机浏览器中访问此地址，可以预览移动端效果。

## 📱 移动端预览

### 方法一：局域网访问

1. 确保手机和电脑在同一 WiFi 网络
2. 在开发服务器启动信息中找到 Network 地址
3. 在手机浏览器中输入该地址

### 方法二：浏览器响应式模式

1. 按 `F12` 打开开发者工具
2. 点击设备工具栏图标（或按 `Ctrl + Shift + M`）
3. 选择设备类型（iPhone、iPad 等）
