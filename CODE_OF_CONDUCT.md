# 社区行为准则

## 我们的承诺

为了建立一个开放和欢迎的社区，我们承诺参与、贡献者应该创建一个没有骚扰的环境的项目和社区。无论年龄、体型、可见性或隐形残疾、种族、民族、性别特征、性别认同和表达、经验水平、教育、社会经济地位、国籍、个人外表、种族、宗教或性认同和取向如何。

## 我们的标准

有助于创造积极环境的行为包括：

- 使用欢迎和包容的语言
- 尊重不同的意见、观点和经验
- 接受建设性的批评
- 关注什么对社区最有利
- 表示对其他社区成员的同情

不可接受的行为包括：

- 使用性暗示的语言或意象
- 嘲笑、羞辱、贬低或对他人的人身攻击
- 骚扰
- 非征求同意地分享他人的私密信息
- 其他可能被合理认为在专业环境中不适当的行为

## 执行

不遵守此代码的社区领导者可能会被社区负责人将其从项目中暂时或永久删除。

如果你看到不可接受的行为，请报告：

- **Email**: your-email@example.com
- **GitHub Issues**: 创建私密应用

所有投诉将被审核和调查，将采取认为必要和适当的行动。
