## 一、完成的功能

### ✅ 1.1 重构 PersonCard 信息结构

**固定 4 行结构**：

1. **第1行**：昵称 + 阶段 Badge + 热度
   - 左侧：昵称（18px，加粗）
   - 右侧：阶段标签 + 热度分数

2. **第2行**：最后联系时间 + 状态提示语
   - 左侧：最后联系时间（相对时间：今天/昨天/X天前）
   - 右侧：状态提示（今天联系过/本周联系/需要关注等）

3. **第3行**：Next Action 行动条（高亮）
   - 渐变紫色背景，视觉突出
   - 显示当前 nextAction
   - 支持操作：
     - ✅ 直接编辑（编辑图标按钮）
     - ✅ 标记完成（完成图标按钮）
     - ✅ AI 生成建议（魔法图标按钮）
   - 如果没有 nextAction，显示"设置行动"按钮

4. **第4行**：快捷操作区
   - "记录互动"按钮（主要操作）
   - "更多"下拉菜单（设提醒、生成建议、查看详情）

### ✅ 1.2 实现 Next Action 行动条功能

#### 功能详情：

1. **直接编辑**
   - 点击编辑按钮 → 打开编辑弹窗
   - 支持输入/编辑 nextAction 内容
   - 支持清空 nextAction

2. **标记完成**
   - 点击完成按钮 → 确认对话框
   - 确认后清空 nextAction

3. **AI 生成建议**
   - 点击 AI 按钮 → 打开记录互动弹窗
   - 提示用户可以在弹窗中使用 AI 功能
   - 结合互动记录生成建议

---

### ✅ 1.3 精简卡片操作

1. **移除显式"查看详情"按钮**
   - 从快捷操作区移除
   - 移到"更多"下拉菜单中

2. **点击卡片空白区域进入详情页**
   - 卡片整体可点击
   - 点击空白区域跳转详情页
   - 点击按钮/链接不触发跳转（使用 `@click.stop`）

### ✅ 1.4 记录互动优化

**之前**：列表页 → 点击"记录互动" → 跳转详情页 → 时间线 Tab → 新增互动 → 打开弹窗（5步）

**现在**：列表页 → 点击"记录互动" → 直接打开弹窗（1步）

**实现**：

- 修改 `handleAddInteraction` 逻辑，不再跳转
