# 项目概览文档

## 📋 项目基本信息

- **项目名称**：溪午听风个人网站
- **项目类型**：全栈个人开发者网站
- **开发语言**：TypeScript (前端) + C# (后端) + Python (AI服务)
- **主要框架**：Nuxt 3 + .NET 8 + FastAPI
- **数据库**：MySQL + 向量数据库 (Chroma)

## 🎯 核心功能模块

### 1. 内容管理系统
- **Markdown 驱动**：所有内容通过 Markdown 文件管理
- **自动路由**：添加 Markdown 文件自动生成页面
- **SEO 优化**：自动生成元数据

### 2. 后台管理系统
- **文章管理**：CRUD 操作
- **项目管理**：项目展示管理
- **工具管理**：工具展示管理
- **访客分析**：数据统计和可视化
- **系统设置**：配置管理

### 3. AI 能力服务
- **聊天对话**：基于 LLM 的对话功能
- **文档处理**：文档解析、分段、摘要
- **知识库问答**：基于 RAG 的问答系统
- **工具函数**：文本摘要、标题生成等

### 4. 访客互动系统
- **访客留言**：弹幕墙、留言板
- **访客足迹**：访问记录、地图展示
- **等级系统**：访客等级和积分
- **互动效果**：特效和动画

## 📊 数据流

### 内容管理流程

```
Markdown 文件 (content/)
    ↓
@nuxt/content 解析
    ↓
自动生成页面路由
    ↓
前端渲染展示
```

### API 调用流程

```
前端组件
    ↓
useApi composable
    ↓
HTTP 请求
    ↓
.NET WebAPI
    ↓
MySQL 数据库
```

### AI 服务调用流程

```
前端组件
    ↓
HTTP 请求
    ↓
.NET WebAPI (代理)
    ↓
Python AI 服务
    ↓
LLM API / 向量数据库
```

## 🔐 安全机制

1. **认证授权**
   - JWT Token 认证
   - 角色权限控制 (RBAC)
   - 路由守卫

2. **数据安全**
   - 参数化查询 (防止 SQL 注入)
   - 输入验证和清理
   - 敏感信息加密

3. **API 安全**
   - 内部 Token 验证 (AI 服务)
   - CORS 配置
   - 请求限流

## 🚀 部署架构

### 开发环境
- **前端**：`npm run dev` (localhost:3000)
- **后端**：`dotnet run` (localhost:5234)
- **AI 服务**：`uvicorn app.main:app` (localhost:8001)

### 生产环境
- **前端**：Nginx 静态文件服务
- **后端**：.NET 应用 (systemd 服务)
- **AI 服务**：Python 应用 (systemd 服务)
- **数据库**：MySQL 8.0
- **反向代理**：Nginx
