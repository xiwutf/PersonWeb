# 移动端最佳打开方式

## 📱 推荐打开方式（按效果排序）

### ⭐ 方式一：添加到主屏幕（PWA 模式）- 最佳体验

这是**最推荐**的方式，可以获得类似原生 App 的体验。

#### 如何添加

**iOS (iPhone/iPad):**
1. 在 Safari 中打开网站
2. 点击底部的**分享按钮**（方框+向上箭头）
3. 选择**"添加到主屏幕"**
4. 自定义名称（可选）
5. 点击**"添加"**

**Android (Chrome/Edge):**
1. 在浏览器中打开网站
2. 点击右上角**菜单**（三个点）
3. 选择**"添加到主屏幕"** 或 **"安装应用"**
4. 确认添加

#### 优势

✅ **全屏体验**：没有浏览器地址栏和工具栏  
✅ **独立图标**：在主屏幕上显示为独立应用  
✅ **快速启动**：点击图标直接打开，无需输入网址  
✅ **沉浸式体验**：更像原生 App  
✅ **离线支持**：如果配置了 Service Worker，可以离线访问  

---

### 方式二：浏览器全屏模式

#### Chrome/Edge (Android)

1. 打开网站
2. 点击右上角**菜单**
3. 选择**"添加到主屏幕"**
4. 会自动以全屏模式打开

#### Safari (iOS)

1. 打开网站
2. 点击底部的**分享按钮**
3. 选择**"添加到主屏幕"**
4. 从主屏幕打开时会自动全屏

---

### 方式三：浏览器普通模式

直接在浏览器中输入网址访问。

**优点：**
- 最简单，无需任何设置
- 可以使用浏览器功能（书签、历史等）

**缺点：**
- 有浏览器地址栏和工具栏
- 体验不如全屏模式

## 🎨 优化移动端体验的配置

### 1. Viewport 配置（已配置）

项目已配置正确的 viewport：

```html
<meta name="viewport" content="width=device-width, initial-scale=1">
```

这确保了：
- 页面宽度自适应设备
- 初始缩放比例为 1:1
- 防止意外缩放

### 2. 主题颜色（建议添加）

可以在 `nuxt.config.ts` 中添加主题颜色，让浏览器地址栏和状态栏颜色与网站主题一致：

```typescript
app: {
  head: {
    meta: [
      // ... 现有配置
      { name: 'theme-color', content: '#3b82f6' }, // 浏览器地址栏颜色
      { name: 'apple-mobile-web-app-capable', content: 'yes' }, // iOS 全屏
      { name: 'apple-mobile-web-app-status-bar-style', content: 'default' }, // iOS 状态栏样式
    ],
    link: [
      // ... 现有配置
      { rel: 'apple-touch-icon', href: '/favicon.png' }, // iOS 主屏幕图标
    ]
  }
}
```

### 3. PWA Manifest（可选，需要安装模块）

如果想获得完整的 PWA 体验，可以安装 `@vite-pwa/nuxt` 模块：

```bash
npm install @vite-pwa/nuxt -D
```

然后在 `nuxt.config.ts` 中配置。

## 🔧 技术实现

### 检查是否支持添加到主屏幕

在浏览器控制台运行：

```javascript
// 检查是否已安装
if (window.matchMedia('(display-mode: standalone)').matches) {
  console.log('已以独立模式运行（从主屏幕打开）')
} else {
  console.log('在浏览器中运行')
}
```

### 检测显示模式

```javascript
// 在组件中检测
const isStandalone = computed(() => {
  if (process.client) {
    return window.matchMedia('(display-mode: standalone)').matches ||
           (window.navigator as any).standalone === true // iOS
  }
  return false
})
```
