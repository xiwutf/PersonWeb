## 📝 使用说明

### 1. 定投计划管理

**创建定投计划**：
1. 进入 `/admin/dca-plan`
2. 点击"新增定投计划"
3. 填写代码、名称、类型、金额、频率等信息
4. 点击"自动获取"获取名称和价格（可选）
5. 保存

**执行定投**：
- 点击"立即执行"按钮手动执行定投
- 系统会自动获取当前价格并更新投资记录

### 2. 价格提醒管理

**创建价格提醒**：
1. 进入 `/admin/price-alert`
2. 点击"新增提醒"
3. 填写代码、目标价格、提醒类型等信息
4. 保存

**刷新价格**：
- 点击"刷新价格"按钮
- 系统会自动检查所有启用的提醒是否触发

### 3. 交易记录导入

**导入步骤**：
1. 进入 `/admin/investment`
2. 点击"选择文件导入"按钮
3. 选择 CSV 文件（格式见下方）
4. 等待导入完成
5. 查看导入结果

**CSV 格式**：
```csv
代码,名称,类型,交易类型,数量,价格,交易日期,备注
005918,天弘沪深300ETF联接C,fund,buy,1000,1.4078,2024-01-01,支付宝购买
000001,平安银行,stock,buy,100,12.50,2024-01-02,
```

### 4. 数据导出

**导出步骤**：
1. 进入 `/admin/investment`
2. 点击"📥 导出数据"按钮
3. 选择要导出的数据类型：
   - 导出投资记录
   - 导出交易记录
   - 导出统计数据
4. 文件会自动下载

## 🔧 技术实现

### 数据库表

- `dca_plan` - 定投计划表
- `dca_execution` - 定投执行记录表
- `price_alert` - 价格提醒表

### API 接口

#### 定投计划
- `GET /api/DcaPlan` - 获取列表
- `POST /api/DcaPlan` - 创建
- `PUT /api/DcaPlan/{id}` - 更新
- `DELETE /api/DcaPlan/{id}` - 删除
- `POST /api/DcaPlan/{id}/execute` - 执行定投

#### 价格提醒
- `GET /api/PriceAlert` - 获取列表
- `POST /api/PriceAlert` - 创建
- `PUT /api/PriceAlert/{id}` - 更新
- `DELETE /api/PriceAlert/{id}` - 删除
- `POST /api/PriceAlert/refresh-prices` - 刷新价格

#### 交易记录导入
- `POST /api/InvestmentImport/import` - 导入文件

#### 数据导出
- `GET /api/InvestmentExport/investments/csv` - 导出投资记录
- `GET /api/InvestmentExport/transactions/csv` - 导出交易记录
- `GET /api/InvestmentExport/stats/csv` - 导出统计数据

#### 历史统计
- `GET /api/Investment/stats/history?period=month&days=30` - 获取历史统计数据

## ⚠️ 注意事项

1. **定时任务**：目前定投计划和价格提醒需要手动执行，定时任务功能待实现
2. **场外基金**：部分场外基金可能无法自动获取价格，需要手动输入
3. **API 限制**：东方财富 API 有频率限制，建议不要过于频繁刷新
4. **Excel 支持**：目前 CSV 导入已实现，Excel 导入需要安装 EPPlus 包
