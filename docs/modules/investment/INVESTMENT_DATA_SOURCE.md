# 投资仪表盘数据来源说明

## 📊 数据来源

### 1. 投资记录数据（真实数据）

**来源**：用户手动添加的投资记录

**存储位置**：数据库 `investment` 表

**包含字段**：
- **代码** (`code`)**: 股票或基金代码（如：000001、510300）
- **名称** (`name`)**: 投资标的名称（如：平安银行、沪深300ETF）
- **类型** (`type`)**: 投资类型（`stock` 股票 或 `fund` 基金）
- **持仓数量** (`quantity`)**: 持有的数量
- **成本价** (`cost_price`)**: 买入时的成本价格
- **总成本** (`total_cost`)**: 总成本 = 持仓数量 × 成本价
- **备注** (`notes`)**: 其他备注信息

**数据真实性**：✅ **真实数据** - 这些数据由您手动输入，完全真实

### 2. 当前价格数据（可能不准确）

**来源**：东方财富 API（`https://push2.eastmoney.com/api/qt/stock/get`）

**获取方式**：
- 点击"刷新价格"按钮时，系统会尝试调用东方财富 API 获取最新价格
- 如果 API 调用失败或未配置，当前价格会显示为 `¥0.00`

**当前状态**：⚠️ **API 未完全实现**
- 代码中已预留了 API 调用接口
- 但价格解析逻辑尚未完成
- 目前返回值为 `0`，导致市值和盈亏显示为负值

**数据真实性**：
- ✅ **API 来源真实**：东方财富是正规的金融数据提供商
- ⚠️ **当前未生效**：由于 API 解析未完成，价格显示为 0

### 3. 计算数据（基于上述数据）

**市值** (`market_value`) = 持仓数量 × 当前价格
- 如果当前价格为 0，市值也会是 0

**盈亏** (`profit_loss`) = 市值 - 总成本
- 如果当前价格为 0，盈亏 = 0 - 总成本 = 负值（显示为亏损）

**盈亏比例** (`profit_rate`) = (盈亏 / 总成本) × 100%
- 如果当前价格为 0，盈亏比例 = -100%

## 🔄 数据更新流程

### 添加投资记录
1. 用户填写：代码、名称、类型、持仓数量、成本价
2. 系统计算：总成本 = 持仓数量 × 成本价
3. 系统尝试获取当前价格（目前返回 0）
4. 系统计算：市值、盈亏、盈亏比例

### 刷新价格
1. 用户点击"刷新价格"按钮
2. 系统遍历所有投资记录
3. 对每条记录调用 `GetCurrentPrice` 方法
4. 尝试从东方财富 API 获取最新价格
5. 更新当前价格、市值、盈亏等字段

## ⚠️ 注意事项

### 当前价格显示为 0 的原因

1. **API 未完全实现**
   - `GetCurrentPrice` 方法中，API 响应解析逻辑被注释掉了
   - 目前直接返回 `0`

2. **API 调用可能失败**
   - 网络问题
   - API 接口变更
   - 请求频率限制

3. **解决方案**
   - **临时方案**：可以手动编辑投资记录，直接输入当前价格
   - **长期方案**：完善 `GetCurrentPrice` 方法，正确解析 API 响应

## 📚 API 文档参考

### 东方财富 API

- **接口地址**：`https://push2.eastmoney.com/api/qt/stock/get`
- **参数说明**：
  - `secid`: 股票代码（需要转换为特定格式）
  - `fields`: 需要获取的字段列表
- **响应格式**：JSON
- **价格字段**：`data.f43`（当前价格）

### 代码格式说明

- **A股股票**：`1.000001`（1. + 6位代码）
- **基金**：`0.510300`（0. + 6位代码）
