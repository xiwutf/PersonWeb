# 项目重复标题修复说明

## 问题描述

项目展示页面出现了重复的项目标题，例如：
- "个人数字资产平台（本网站）" 和 "个人网站系统"
- "AI 创作助手" 和 "AI 智能助手"
- "访客分析系统（Analytics）" 和 "访客分析系统"

## 解决方案

### 1. 前端去重（已实现）

在 `pages/projects/index.vue` 中添加了去重逻辑：

1. **完全相同的标题去重**：根据标题（标准化后）去重，保留最早创建的版本
2. **相似标题去重**：处理已知的重复项映射：
   - "个人数字资产平台" 优先于 "个人网站系统"、"个人网站 V2"
   - "AI 创作助手" 优先于 "AI 智能助手"
   - "访客分析系统（Analytics）" 优先于 "访客分析系统"

### 2. 数据库清理脚本

创建了两个 SQL 脚本：

#### `database/check_duplicate_projects.sql`
用于检查数据库中的重复项目：
- 查找完全相同的标题
- 查找标题相似的项目
- 统计项目总数和唯一标题数

#### `database/remove_duplicate_projects.sql`
用于清理数据库中的重复项目：
- 删除 "个人网站 V2"
- 删除 "访客分析系统"（保留 "访客分析系统（Analytics）"）
- 删除 "个人网站系统"（如果存在 "个人数字资产平台"）
- 删除 "AI 智能助手"（如果存在 "AI 创作助手"）

## 使用方法

### 检查重复项目

```bash
mysql -u your_username -p your_database_name < database/check_duplicate_projects.sql
```

### 清理重复项目（谨慎操作）

**⚠️ 注意：执行前请先备份数据库！**

```bash
mysql -u your_username -p your_database_name < database/remove_duplicate_projects.sql
```

## 已知的重复项映射

| 保留的标题 | 要删除的标题 |
|-----------|------------|
| 个人数字资产平台（本网站） | 个人网站系统、个人网站 V2 |
| AI 创作助手 | AI 智能助手 |
| 访客分析系统（Analytics） | 访客分析系统 |

## 前端去重逻辑说明

1. **标准化标题**：去除空格、转换为小写，用于比较
2. **完全匹配去重**：如果标题完全相同，保留最早创建的版本
3. **相似标题处理**：根据预定义的重复项映射，排除重复的标题

## 注意事项

1. **数据备份**：执行删除脚本前，请务必备份数据库
2. **手动确认**：某些相似标题可能需要手动确认是否为重复项
3. **前端缓存**：清理数据库后，可能需要清除浏览器缓存才能看到效果
