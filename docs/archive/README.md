# 归档说明

`docs/archive/` 用于存放不再进入 AI 主执行路径的文档。

这些文档仍可能有参考价值，但不再属于默认必读集合。

## 当前归档分类

- `legacy/`：本轮极简化收缩后移出的低频目录、阶段性文档、历史专题资料
- `ai-worklogs/`：AI 协作过程文档、执行清单、阶段操作记录
- `style-history/`：早期样式审计、样式改造方案、历史扫描结果

## 使用规则

- AI 默认先读 `AGENTS.md` 和 `docs/` 当前保留的核心手册
- 只有当核心手册不足以支持任务时，才继续查阅 `archive/`
- 历史方案、阶段报告、旧专题索引，统一放在 `archive/legacy/`

## 当前保留的归档辅助文件

- `DEPRECATION_AUDIT_TODO.md`：后续文件清理待办

## 归档原则

满足任一条件时，优先归档：

- 文档是一次性执行记录
- 文档描述的是某个历史阶段而不是当前规范
- 文档内容已被更精简的核心手册替代
- 文档不再适合作为 AI 默认执行入口
