# 样式系统最佳实践

## 一、核心理念

### 1.1 设计原则
1. **不硬编码任何视觉值**
   - 颜色必须使用变量
   - 尺寸必须使用变量
   - 间距必须使用变量
   - 圆角必须使用变量

2. **分层设计**
   - 基础层：具体数值
   - 语义层：功能描述
   - 组件层：组件特定

3. **单一数据源**
   - 所有样式定义在 CSS 变量中
   - 避免重复定义
   - 易于维护和更新

### 1.2 变量使用优先级
```
组件变量 > 语义变量 > 基础变量
```

## 二、具体实践

### 2.1 CSS 开发规范

#### ✅ 正确做法
```css
.button {
  /* 使用语义变量 */
  background: var(--btn-bg-primary);
  color: var(--btn-text-primary);
  border-radius: var(--btn-radius);

  /* 使用组件变量 */
  height: var(--btn-height-md);
  padding: 0 var(--btn-padding-x-md);

  /* 使用基础变量 */
  transition: var(--transition);
}
```

#### ❌ 错误做法
```css
.button {
  /* 直接写死值 */
  background: #3b82f6;
  color: white;
  border-radius: 8px;
  height: 40px;
  padding: 0 16px;
  transition: all 0.2s ease;
}
```

### 2.2 Vue 组件开发

#### ✅ 正确做法
```vue
<template>
  <button class="btn btn--primary" :disabled="loading">
    <span v-if="loading" class="btn__loading"></span>
    <slot>{{ label }}</slot>
  </button>
</template>

<script setup>
const props = defineProps({
  loading: Boolean,
  label: String
})
</script>

<style scoped>
.btn {
  /* 使用基础样式 */
  display: inline-flex;
  align-items: center;
  justify-content: center;
  gap: var(--spacing-2);

  /* 使用组件变量 */
  height: var(--btn-height-md);
  padding: 0 var(--btn-padding-x-md);
  border-radius: var(--btn-radius);
  font-weight: var(--btn-font-weight);

  /* 使用语义变量 */
  background: var(--btn-bg-primary);
  color: var(--btn-text-primary);
  border: 1px solid var(--btn-bg-primary);

  transition: var(--transition);
}

.btn--primary {
  /* 主要按钮样式已在变量中定义 */
}

.btn--loading {
  /* 加载状态样式 */
}
</style>
```

### 2.3 Tailwind CSS 集成

#### 在 tailwind.config.ts 中配置
```typescript
theme: {
  extend: {
    colors: {
      // 使用语义变量
      'bg-card': 'var(--color-bg-card)',
      'text-primary': 'var(--color-text-primary)',
      'border-default': 'var(--color-border-default)',
      'primary': 'var(--color-primary)',

      // 或使用基础颜色
      'blue-500': 'var(--color-blue-500)',
    },
    spacing: {
      // 使用间距变量
      '4': 'var(--spacing-4)',
      '6': 'var(--spacing-6)',
    },
    borderRadius: {
      // 使用圆角变量
      'md': 'var(--radius-md)',
      'lg': 'var(--radius-lg)',
    },
    boxShadow: {
      // 使用阴影变量
      'sm': 'var(--shadow-sm)',
      'md': 'var(--shadow-md)',
    },
  }
}
```

#### 在 Vue 中使用
```vue
<template>
  <div class="bg-bg-card border border-border-default rounded-lg p-6 shadow-md">
    <h1 class="text-2xl font-semibold text-primary">标题</h1>
    <p class="text-secondary mt-2">描述文字</p>
  </div>
</template>
```

### 2.4 响应式设计

#### ✅ 使用 CSS 变量实现响应式
```css
:root {
  --container-width: 1200px;
  --header-height: 60px;
}

@media (max-width: 768px) {
  :root {
    --container-width: 100%;
    --header-height: 50px;
  }
}

.container {
  max-width: var(--container-width);
  margin: 0 auto;
}

.header {
  height: var(--header-height);
}
```

### 2.5 主题切换

#### 使用 JavaScript 切换主题
```javascript
// 在 composables/useTheme.ts 中
export function useTheme() {
  const theme = ref('light')

  const setTheme = (newTheme) => {
    theme.value = newTheme
    document.documentElement.setAttribute('data-theme', newTheme)
    localStorage.setItem('theme', newTheme)
  }

  const toggleTheme = () => {
    const themes = ['light', 'dark', 'tech-blue']
    const currentIndex = themes.indexOf(theme.value)
    const nextIndex = (currentIndex + 1) % themes.length
    setTheme(themes[nextIndex])
  }

  return {
    theme,
    setTheme,
    toggleTheme
  }
}
```

#### 在组件中使用
```vue
<template>
  <button @click="toggleTheme">
    切换主题
  </button>
</template>

<script setup>
import { useTheme } from '~/composables/useTheme'

const { toggleTheme } = useTheme()
</script>
```

## 三、常见问题解决方案

### 3.1 第三方组件样式覆盖

```css
/* 覆盖第三方组件样式 */
.my-component {
  /* 使用变量覆盖 */
  --component-bg: var(--color-bg-card);
  --component-border: var(--color-border-default);
  --component-text: var(--color-text-primary);

  background: var(--component-bg);
  border: 1px solid var(--component-border);
  color: var(--component-text);
}
```

### 3.2 渐变背景处理

```css
/* 使用变量创建渐变 */
.gradient-primary {
  background: linear-gradient(135deg,
    var(--color-primary) 0%,
    var(--color-primary-hover) 100%
  );
}

/* 使用透明度 */
.gradient-soft {
  background: linear-gradient(135deg,
    var(--color-primary-soft) 0%,
    var(--color-primary-soft) 100%
  );
}
```

### 3.3 动画和过渡

```css
/* 使用变量定义动画 */
@keyframes fadeIn {
  from {
    opacity: 0;
    transform: translateY(var(--spacing-4));
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

.fade-in {
  animation: fadeIn var(--transition-slower) ease-out;
}
```

## 四、性能优化

### 4.1 减少重排重绘

```css
/* 使用 transform 和 opacity 代替改变位置和大小 */
.card {
  transform: translateZ(0); /* 触发硬件加速 */
  will-change: transform, opacity;
}

/* 避免使用 width/height 动画 */
.card__hover {
  transform: scale(1.02);
}
```

### 4.2 CSS 选择器优化

```css
/* ✅ 高效选择器 */
.btn { /* ... */ }
.btn-primary { /* ... */ }

/* ❌ 低效选择器 */
.page .header .nav .nav-item .btn { /* ... */ }
```

### 4.3 变量性能考虑

```css
/* ✅ 在 :root 中定义变量 */
:root {
  --primary-color: #3b82f6;
}

/* ❌ 在元素内部定义变量 */
.btn {
  --primary-color: #3b82f6; /* 会降低性能 */
}
```
