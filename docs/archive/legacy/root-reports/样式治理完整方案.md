## 三、变量体系详解

### 3.1 变量结构
```
theme.css (已存在)
├── 基础颜色定义
├── 主题变量定义
└── 模块主题覆盖

theme-variables.css (新增)
├── 基础颜色层
├── 语义颜色层
├── 组件变量层
├── 尺寸系统
├── 动画系统
└── 通用类

component-styles.css (新增)
├── 按钮组件
├── 输入框组件
├── 标签组件
├── 卡片组件
├── 表格组件
├── 分页组件
├── 提示组件
├── 空状态组件
├── 加载组件
└── 徽章组件
```

### 3.2 变量命名规范

#### 基础颜色
```css
/* 格式：color-{hue}-{level} */
color-blue-500
color-red-600
color-gray-200
```

#### 语义颜色
```css
/* 格式：color-{purpose} */
color-primary
color-bg-card
color-text-primary
color-border-default
```

#### 组件变量
```css
/* 格式：{component}-{state}-{prop} */
btn-bg-primary
input-height-md
card-radius
```

#### 尺寸变量
```css
/* 间距 */
spacing-4: 16px

/* 圆角 */
radius-md: 8px

/* 阴影 */
shadow-md: 0 4px 6px -1px rgba(0, 0, 0, 0.1)

/* 字体 */
text-lg: 1.125rem
font-semibold: 600
```

## 五、使用示例

### 5.1 按钮组件使用
```vue
<template>
  <button class="btn btn--primary">
    <slot>按钮</slot>
  </button>
</template>

<style scoped>
.btn {
  /* 使用组件变量 */
  height: var(--btn-height-md);
  padding: 0 var(--btn-padding-x-md);
  border-radius: var(--btn-radius);

  /* 使用语义变量 */
  background: var(--btn-bg-primary);
  color: var(--btn-text-primary);

  transition: var(--transition);
}

.btn--primary {
  background: var(--color-primary);
  border: none;
}

.btn--primary:hover {
  background: var(--color-primary-hover);
}
</style>
```

### 5.2 卡片组件使用
```vue
<template>
  <div class="card card--lg">
    <div class="card__header">
      <h3 class="card__title">标题</h3>
    </div>
    <div class="card__content">
      <p>内容</p>
    </div>
  </div>
</template>

<style scoped>
.card {
  /* 使用主题变量 */
  background: var(--card-bg);
  border: 1px solid var(--card-border);
  border-radius: var(--card-radius);
  box-shadow: var(--card-shadow-sm);

  transition: var(--transition);
}

.card--lg {
  padding: var(--card-padding-lg);
}
</style>
```

### 5.3 自定义样式使用
```css
/* 使用基础变量 */
.custom-widget {
  background: var(--color-bg-card);
  border: 1px solid var(--color-border-default);
  border-radius: var(--radius-lg);

  /* 使用尺寸系统 */
  padding: var(--spacing-6);
  margin-bottom: var(--spacing-4);
}

/* 使用组件变量 */
.custom-button {
  /* 继承按钮基础样式 */
  height: var(--btn-height-md);
  padding: 0 var(--btn-padding-x-md);
  border-radius: var(--btn-radius);

  /* 覆盖特定样式 */
  background: var(--color-primary);
  color: var(--color-primary-text);
}
```

## 六、维护指南

### 6.1 添加新变量
1. **确定层级**
   - 基础颜色：添加到 `theme-variables.css`
   - 语义颜色：添加到语义层
   - 组件变量：添加到组件变量层

2. **命名规范**
   - 使用语义化命名
   - 保持一致性
   - 添加注释说明

### 6.2 添加新组件
1. **参考现有组件**
   - 复制基础样式
   - 修改特定变量

2. **完整状态**
   - hover、active、disabled
   - loading、error 等状态

### 6.3 主题维护
1. **新主题添加**
   ```css
   [data-theme='new-theme'] {
     --color-primary: #new-value;
     /* 其他变量 */
   }
   ```

2. **主题测试**
   - 测试所有组件显示
   - 检查对比度
   - 验证可读性

## 七、问题排查

### 7.1 变量不生效
1. 检查 CSS 文件引入顺序
2. 确认变量作用域
3. 检查优先级覆盖

### 7.2 主题切换问题
1. 确认 `data-theme` 属性设置
2. 检查变量覆盖规则
3. 验证 JavaScript 逻辑

### 7.3 样式冲突
1. 使用浏览器 DevTools 检查
2. 检查选择器优先级
3. 使用 `!important` 谨慎
