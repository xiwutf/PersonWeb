# 东方财富 API 使用指南

## 📌 重要说明

**东方财富 API 是公开接口，无需注册或获取 API Key！**

我们使用的接口是东方财富提供的公开数据接口，可以直接调用，不需要任何认证。

## 🔗 API 接口地址

```
https://push2.eastmoney.com/api/qt/stock/get
```

## 📋 接口参数说明

### 必需参数

- **secid**: 证券代码（格式：`市场代码.股票代码`）
  - A股股票：`1.000001`（1. + 6位代码，不足6位前面补0）
  - 基金：`0.510300`（0. + 6位代码，不足6位前面补0）
  
- **fields**: 需要获取的字段列表（用逗号分隔）
  - `f43`: 当前价格（单位：分，需要除以100）
  - `f57`: 股票代码
  - `f58`: 股票名称
  - 更多字段请参考下方字段说明

### 示例请求

**A股股票（平安银行 000001）**：
```
https://push2.eastmoney.com/api/qt/stock/get?secid=1.000001&fields=f43,f57,f58
```

**基金（沪深300ETF 510300）**：
```
https://push2.eastmoney.com/api/qt/stock/get?secid=0.510300&fields=f43,f57,f58
```

## 📊 响应格式

### 成功响应示例

```json
{
  "rc": 0,
  "rt": 6,
  "svr": 182998821,
  "lt": 1,
  "full": 1,
  "data": {
    "f43": 1250,  // 当前价格（单位：分，实际价格 = 1250 / 100 = 12.50）
    "f57": "000001",
    "f58": "平安银行"
  }
}
```

### 字段说明

| 字段代码 | 说明 | 单位 |
|---------|------|------|
| f43 | 当前价格 | 分（需要除以100） |
| f57 | 股票代码 | - |
| f58 | 股票名称 | - |
| f44 | 最高价 | 分 |
| f45 | 最低价 | 分 |
| f46 | 今开 | 分 |
| f47 | 昨收 | 分 |
| f48 | 涨跌额 | 分 |
| f60 | 涨跌幅 | %（需要除以100） |
| f107 | 成交量 | 手 |
| f137 | 成交额 | 元 |
| f170 | 换手率 | %（需要除以100） |

## ⚠️ 注意事项

### 1. 代码格式要求

- **A股股票**：必须使用 `1.` 前缀
  - 正确：`1.000001`（平安银行）
  - 错误：`000001` 或 `1.1`
  
- **基金**：必须使用 `0.` 前缀
  - 正确：`0.510300`（沪深300ETF）
  - 错误：`510300` 或 `0.5103`

- **代码补零**：代码必须补足6位
  - 正确：`1.000001`
  - 错误：`1.1` 或 `1.001`

### 2. 价格单位

API 返回的价格单位是**分**，需要除以 100 才能得到实际价格。

例如：
- API 返回：`f43: 1250`
- 实际价格：`1250 / 100 = 12.50` 元

### 3. 请求频率限制

虽然这是公开接口，但建议：
- 不要过于频繁地请求（建议间隔 1-2 秒）
- 批量刷新价格时，可以添加延迟避免被封禁
- 如果遇到 429 错误（请求过多），请降低请求频率

### 4. 错误处理

如果 API 调用失败或返回错误，系统会：
- 记录错误日志
- 返回价格 `0`，表示获取失败
- 不会影响其他功能的正常使用

## 🔄 代码中的实现

在 `InvestmentController.cs` 中的 `GetCurrentPrice` 方法已经实现了：

1. **自动格式转换**：
   - 股票代码自动补零到6位
   - 根据类型（stock/fund）自动添加前缀（1./0.）

2. **价格解析**：
   - 自动从响应中提取 `f43` 字段
   - 自动除以 100 转换为实际价格

3. **错误处理**：
   - 网络错误时返回 0
   - 解析错误时返回 0
   - 记录错误日志便于调试

## 🚀 使用示例

### 在投资仪表盘中刷新价格

1. 打开投资仪表盘页面
2. 点击"刷新价格"按钮
3. 系统会自动：
   - 遍历所有投资记录
   - 根据代码和类型构建正确的 `secid`
   - 调用东方财富 API 获取最新价格
   - 更新数据库中的价格、市值、盈亏等信息

### 添加新投资记录时

1. 填写股票/基金代码（如：`000001`）
2. 选择类型（股票/基金）
3. 保存时系统会自动：
   - 调用 API 获取当前价格
   - 计算市值和盈亏

## 🔧 如果 API 不可用

如果东方财富 API 不可用或返回错误，可以考虑以下替代方案：

### 方案1：手动输入价格

在编辑投资记录时，可以手动输入当前价格，系统会基于手动输入的价格计算市值和盈亏。

### 方案2：使用其他数据源

- **新浪财经 API**：`http://hq.sinajs.cn/list=`
- **腾讯财经 API**：`http://qt.gtimg.cn/q=`
- **雪球 API**：需要注册和认证

### 方案3：使用付费 API

如果需要更稳定和准确的数据，可以考虑使用付费的金融数据 API：
- 聚宽
- 米筐
- Tushare（需要注册）

## 📝 常见问题

### Q: 为什么价格显示为 0？

A: 可能的原因：
1. API 调用失败（网络问题）
2. 股票代码格式错误
3. 股票代码不存在
4. API 服务暂时不可用

**解决方法**：
- 检查网络连接
- 确认股票代码格式正确（6位数字）
- 可以手动编辑投资记录输入价格

### Q: 基金代码如何获取？

A: 基金代码通常是6位数字，例如：
- 沪深300ETF: `510300`
- 中证500ETF: `510500`
- 创业板ETF: `159915`

可以在东方财富网、天天基金网等网站查询。

### Q: API 调用失败怎么办？

A: 
1. 检查网络连接
2. 查看后端日志了解具体错误
3. 可以手动编辑投资记录输入价格
4. 等待一段时间后重试
