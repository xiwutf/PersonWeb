**实现**：

### ✅ 1.2 AI 输出统一格式

#### 统一输出结构：

```typescript
{
  statusHint: string,        // 状态提示语（短）
  nextActions: Array<{       // Next Action 建议（1-3条）
    title: string,
    when: string,
    why: string
  }>
}
```

#### 状态提示语来源：

1. **优先使用 AI 的一句话总结**（如果有）
2. **否则根据未联系天数生成**：
   - ≥ 30 天：`已 X 天未联系，建议关注关系状态`
   - ≥ 14 天：`已 X 天未联系，建议保持联系`
   - ≥ 7 天：`已 X 天未联系，建议适当互动`
   - < 7 天：`建议保持持续互动` 或 `当前有下一步行动，建议按计划推进`

#### Next Action 建议：

- 从 AI 响应中提取
- 限制为最多 3 条
- 每条包含：title（标题）、when（时间建议）、why（原因）

---

### ✅ 1.3 明确 AI 是"建议"

3. **提示**：统一使用"建议"、"可参考"
4. **Next Action 标签**：`建议的下一步行动（可参考）：`

## 三、技术实现

### 3.1 PersonCard 增强

**新增功能**：
- AI 建议卡片展示
- 长时间未联系警告
- expose 方法供父组件调用：
  - `showSuggestion(suggestion)`
  - `setAiLoading(loading)`
  - `setAiError(error)`
  - `hideSuggestion()`

### 3.2 列表页 AI 建议逻辑

**新增函数**：
- `generateLightweightAiSuggestion(personId, autoTrigger)`
  - 获取最近互动记录
  - 调用 AI API
  - 提取轻量级建议
  - 显示在卡片上

- `extractStatusHint(response, person)`
  - 提取或生成状态提示语

- `extractNextActions(response)`
  - 提取 Next Action 建议（最多3条）

- `handleApplyAiSuggestion(personId, action)`
  - 应用 AI 建议到 nextAction

### 3.3 保存互动后的自动建议

**流程**：
1. 保存互动成功
2. 记录 personId
3. 刷新列表
4. 弹出提示："是否查看 AI 建议？"
5. 用户选择：
   - "查看建议" → 生成并显示建议
   - "稍后" → 不做任何操作
