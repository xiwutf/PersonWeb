# 样式变量使用指南

## 一、变量体系说明

### 1.1 变量分层结构

```
主题变量体系
├── 基础颜色层（基础色值）
│   ├── color-{base}-{hue}-{level}  如：color-blue-500
│   └── color-{hue}-{level}         如：blue-500
├── 语义颜色层（语义化命名）
│   ├── color-{purpose}             如：color-primary
│   └── color-{element}-{state}     如：color-bg-card
└── 组件变量层（组件特定）
    ├── {component}-{state}-{prop}  如：button-primary-bg
    └── {element}-{size}-{prop}     如：modal-radius
```

### 1.2 使用优先级

1. **优先使用组件变量**（最高优先级）
   ```css
   .button {
     background: var(--btn-bg-primary);
     color: var(--btn-text-primary);
   }
   ```

2. **其次使用语义变量**
   ```css
   .card {
     background: var(--color-bg-card);
     border-color: var(--color-border-default);
   }
   ```

3. **最后使用基础颜色变量**（最低优先级）
   ```css
   .custom-badge {
     background: var(--color-blue-500);
   }
   ```

## 二、使用规范

### 2.1 CSS 中使用

```css
/* 推荐：使用语义变量 */
.my-component {
  background: var(--color-bg-card);
  color: var(--color-text-primary);
  border-radius: var(--radius-md);
  box-shadow: var(--shadow-sm);
}

/* 不推荐：直接写死值 */
.my-component {
  background: #ffffff;
  color: #0f172a;
  border-radius: 8px;
  box-shadow: 0 1px 2px 0 rgba(0, 0, 0, 0.05);
}
```

### 2.2 Vue 组件中使用

```vue
<template>
  <div class="my-component">
    <h2 class="title">{{ title }}</h2>
    <p class="description">{{ description }}</p>
  </div>
</template>

<style scoped>
.my-component {
  /* 使用变量 */
  background: var(--color-bg-card);
  border: 1px solid var(--color-border-default);
  border-radius: var(--radius-lg);
  padding: var(--spacing-6);

  /* 使用 Tailwind + 变量 */
  @apply bg-bg-card border border-border-default rounded-lg p-6;
}

.title {
  color: var(--color-text-primary);
  font-size: var(--text-2xl);
  font-weight: var(--font-semibold);
  margin-bottom: var(--spacing-4);
}

.description {
  color: var(--color-text-secondary);
  font-size: var(--text-base);
  line-height: var(--leading-relaxed);
}
</style>
```

### 2.3 SCSS 中使用

```scss
// 创建 mixin 使用变量
@mixin card-base {
  background: var(--color-bg-card);
  border: 1px solid var(--color-border-default);
  border-radius: var(--radius-lg);
  box-shadow: var(--shadow-sm);
  padding: var(--spacing-6);
}

.my-card {
  @include card-base;

  &:hover {
    box-shadow: var(--shadow-md);
    border-color: var(--color-border-strong);
  }
}
```

### 2.4 Tailwind CSS 自定义

在 `tailwind.config.ts` 中添加：

```typescript
theme: {
  extend: {
    colors: {
      // 使用语义变量
      'bg-card': 'var(--color-bg-card)',
      'text-primary': 'var(--color-text-primary)',
      'border-default': 'var(--color-border-default)',

      // 或直接使用基础颜色
      'blue-500': 'var(--color-blue-500)',
    },
    spacing: {
      // 使用间距变量
      '4': 'var(--spacing-4)',
      '6': 'var(--spacing-6)',
    },
    borderRadius: {
      // 使用圆角变量
      'md': 'var(--radius-md)',
      'lg': 'var(--radius-lg)',
    },
    boxShadow: {
      // 使用阴影变量
      'sm': 'var(--shadow-sm)',
      'md': 'var(--shadow-md)',
    },
  }
}
```

## 三、最佳实践

### 3.1 组件样式封装

```vue
<!-- 推荐：创建可复用的组件 -->
<template>
  <div class="app-button" :class="buttonClass">
    <slot />
  </div>
</template>

<style scoped>
.app-button {
  /* 使用按钮组件变量 */
  height: var(--btn-height-md);
  padding: 0 var(--btn-padding-x-md);
  border-radius: var(--btn-radius);
  font-weight: var(--btn-font-weight);
  border: none;
  cursor: pointer;
  transition: var(--transition);

  /* 根据类型应用不同样式 */
  &--primary {
    background: var(--btn-bg-primary);
    color: var(--btn-text-primary);

    &:hover {
      background: var(--btn-bg-primary-hover);
    }
  }

  &--secondary {
    background: var(--btn-bg-secondary);
    color: var(--btn-text-secondary);
    border: 1px solid var(--btn-border-secondary);

    &:hover {
      background: var(--btn-bg-secondary-hover);
    }
  }
}
</style>
```

### 3.2 状态管理

```css
/* 使用状态变量 */
.status-success {
  background: var(--color-success-bg);
  border: 1px solid var(--color-success-border);
  color: var(--color-success-text);
}

.status-warning {
  background: var(--color-warning-bg);
  border: 1px solid var(--color-warning-border);
  color: var(--color-warning-text);
}

.status-error {
  background: var(--color-error-bg);
  border: 1px solid var(--color-error-border);
  color: var(--color-error-text);
}
```

### 3.3 响应式设计

```css
/* 使用 CSS 变量实现响应式 */
.container {
  width: 100%;
  max-width: var(--container-max-width);
  margin: 0 auto;
  padding: 0 var(--page-padding-x);
}

@media (max-width: 768px) {
  :root {
    --page-padding-x: var(--spacing-md);
  }
}
```

## 四、常见错误

### 4.1 直接写死值

```css
/* 错误 */
.button {
  background: #3b82f6;
  color: white;
  border-radius: 8px;
}

/* 正确 */
.button {
  background: var(--color-primary);
  color: var(--color-primary-text);
  border-radius: var(--radius-md);
}
```

### 4.2 混合使用变量和写死值

```css
/* 错误 */
.button {
  background: var(--color-primary);
  color: white;  /* 应该使用变量 */
  border-radius: 8px;  /* 应该使用变量 */
}

/* 正确 */
.button {
  background: var(--color-primary);
  color: var(--color-primary-text);
  border-radius: var(--radius-md);
}
```

### 4.3 过度嵌套的选择器

```css
/* 错误：过度嵌套 */
.app .app-header .app-header-nav .app-header-nav-item {
  color: var(--color-text-primary);
}

/* 正确：使用类 */
.nav-item {
  color: var(--color-text-primary);
}
```

## 五、调试技巧

## 六、主题切换

### 6.1 使用 data-theme 属性

```html
<!-- 在 HTML 根元素设置 -->
<html data-theme="dark">
```

### 6.2 通过 JavaScript 切换

```javascript
// 切换主题
function setTheme(theme) {
  document.documentElement.setAttribute('data-theme', theme)
  localStorage.setItem('theme', theme)
}

// 获取当前主题
function getCurrentTheme() {
  return document.documentElement.getAttribute('data-theme') || 'light'
}
```

### 6.3 主题变量覆盖

```css
/* dark theme */
[data-theme='dark'] {
  --color-bg-page: #0f172a;
  --color-bg-card: rgba(255, 255, 255, 0.95);
  --color-text-primary: #f1f5f9;
}

/* light theme */
[data-theme='light'] {
  --color-bg-page: #f8fafc;
  --color-bg-card: #ffffff;
  --color-text-primary: #0f172a;
}
```

## 七、检查清单

开发新样式时，检查以下项目：

- [ ] 是否使用了主题变量而不是硬编码值
- [ ] 变量命名是否符合语义化
- [ ] 是否考虑了不同主题的适配
- [ ] 样式是否具有足够的可维护性
- [ ] 是否创建了可复用的组件
- [ ] 是否添加了适当的过渡动画
