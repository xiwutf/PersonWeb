## 🎯 使用建议

### 第一次配置开发环境
1. 先看 [快速开始指南](./QUICK_START.md)
2. 运行一键配置脚本
3. 按照 [环境检查清单](./ENVIRONMENT_CHECKLIST.md) 验证
4. 遇到问题查看 [开发环境配置指南](./DEVELOPMENT_SETUP.md)

### 换电脑时
1. 运行一键配置脚本
2. 配置环境变量和数据库
3. 按照检查清单验证

### 遇到问题时
1. 查看 [开发环境配置指南](./DEVELOPMENT_SETUP.md) 的"常见问题"部分
2. 查看 `docs/troubleshooting/` 目录下的故障排除文档
