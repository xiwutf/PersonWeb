# 内容更新指南

## 1. 博客文章 (Blog)
位置：`content/blog/`
格式：Markdown (`.md`)

### 新建文章
1. 在 `content/blog/` 目录下创建一个新的 `.md` 文件。
2. 文件名建议使用英文或拼音，例如 `my-new-post.md`。
3. 文件开头必须包含 Frontmatter (元数据)：

```yaml
---
title: 我的新文章
date: 2024-05-20
summary: 这是文章的摘要...
tags: [Vue, Nuxt, 教程]
cover: /images/blog/cover.jpg
---
```

### 撰写内容
在 Frontmatter 下方撰写正文，支持标准 Markdown 语法。
