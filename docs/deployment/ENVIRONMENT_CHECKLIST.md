# 开发环境检查清单

在新电脑上配置开发环境时，按照此清单逐项检查。

## ✅ 前置检查

- [ ] Node.js 18+ 已安装 (`node --version`)
- [ ] npm 已安装 (`npm --version`)
- [ ] .NET SDK 8.0+ 已安装 (`dotnet --version`)
- [ ] MySQL 已安装并运行 (`mysql --version`)
- [ ] Git 已安装 (`git --version`)

## ✅ 项目配置

- [ ] 项目已克隆到本地
- [ ] 前端依赖已安装 (`npm install`)
- [ ] 后端依赖已恢复 (`dotnet restore`)
- [ ] `.env` 文件已创建并配置
- [ ] `appsettings.Development.json` 已创建并配置

## ✅ 数据库配置

- [ ] 数据库已创建 (`personal_site`)
- [ ] 数据库表结构已执行（`database/all_tables.sql`）
- [ ] 数据库连接测试成功
- [ ] 可以正常查询数据

## ✅ 服务启动

- [ ] 后端服务可以启动 (`dotnet run`)
- [ ] 后端 API 文档可以访问 (http://localhost:5234/swagger)
- [ ] 前端服务可以启动 (`npm run dev`)
- [ ] 前端页面可以访问 (http://localhost:3000)
- [ ] 前端可以正常调用后端 API

## ✅ 功能验证

- [ ] 可以访问首页
- [ ] 可以访问后台登录页
- [ ] 可以登录后台
- [ ] 可以查看文章列表
- [ ] 可以创建/编辑文章
- [ ] 可以上传图片
