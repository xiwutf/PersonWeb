# 更新日志

所有本项目的重要变化都会记录在这个文件中。

这个项目遵循 [Semantic Versioning](https://semver.org/)。

## [1.0.0] - 2026-03-09

### ✨ 新增

- 🎉 开源发布！使用 MIT License
- 📱 完整的响应式设计支持
- 🎨 深色/浅色主题切换
- 📝 基于 Markdown 的内容管理系统
- 🛠️ 插件工具展示模块
- 📚 技术博客系统
- 🎯 项目展示模块
- 🔍 SEO 优化
- 📊 ECharts 数据可视化
- 🎮 交互式组件和动画

### 🏗️ 技术栈

- **前端框架**: Nuxt 3 + Vue 3
- **样式**: Tailwind CSS + SCSS
- **UI 库**: Naive UI
- **内容管理**: @nuxt/content
- **数据可视化**: ECharts
- **后端**: .NET 8.0
- **数据库**: MySQL 8.0

### 📝 文档

- ✅ 完整的 README 文档
- ✅ 开发指南
- ✅ 部署指南
- ✅ API 文档
- ✅ 贡献指南

### 🚀 特性

#### 内容系统
- Markdown 驱动的内容管理
- 自动化页面生成
- 分类和标签系统
- 搜索功能

#### UI/UX
- 响应式布局（桌面、平板、手机）
